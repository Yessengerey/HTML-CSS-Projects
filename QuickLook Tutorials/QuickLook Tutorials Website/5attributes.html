Attributes

Attributes are an extra piece of information added inside of a tag. And as was mentioned earlier in this tutorial's Introduction, Attributes provide specific properties/behaviour for that particular content enclosed with that tag via additional embedded technologies. All attributes are usually declared in the opening tag, however self-closing tags may also have some attributes. Furthermore, if following the rules of XHTML - all attributes must have a certain value. Some examples of attributes usage are are given below:

<tag attribute="value"some content</tag>

<img src="name_of_image.img" alt="alternative_name" />

checked="checked"

Some common attributes and their explanation are shown in the table below. Those attributes can be applied to any tags that are inside of the <body> section.

AttributesExplanation
<id>Used to give a specific element a unique identification for CSS.
<class>Used to associate several elements of the web document to a CSS class.
<title>This attribute is used by the screen readers.
It simply provides general information about the element that it's assigned to.
<style>Through this attribute, the element it's assigned to is given certain inline CSS specifications.

Tag-level Association

This is simply a way of organizing elements within the web page for the CSS and other embedded technologies. This kind of 'grouping' helps to assign certain behaviour to several elements at once.

In terms of the Class attribute - there is not limits in its usage. Several tags can be associated with the same Class attribute, as well as a single tag is allowed to have been assigned several Class attributes. Also any individual class attribute can be reused anywhere within a page and in tags of different types.

When it comes to the usage of id attributes, unfortunately there are some restrictions. The id attribute is a unique attribute and so any individual id can only be used once per page. In addition, an element is allowed to have only one id attribute, which means you are not allowed to group several elements under the same id for the CSS.

For example:

<p class="pizza">Pizzas are very delicious</p>

<p class="pizza" id="margherita">Pizza Margherita is the best one</p>
