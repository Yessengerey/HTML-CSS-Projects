Special Characters

To be honest there is not much to learn in terms of Special Characters. The only thing you would want to remember are the entity names and numeric references for the characters. The reason why XHTML requires proper specification for all of them is because some characters are used in the build-up of the content's structure like the "less than" sign or the quotation marks. So if you intend to those kind of characters to actually be visible in the web page, then you'd have to use the special coding for them.

Below are given the most commonly used special characters:
