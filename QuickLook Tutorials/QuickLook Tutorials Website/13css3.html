CSS3

CSS3 introduces some new concepts that not only are interesting but also that enhance the experience for the users of a website.

Box Model

CSS3 brings in a new attribute for the creation of boxes and borders - border-radius and box-shadow.

#box {border-radius: 10px;} - makes rounded up corners for the borders

#box {box-shadow: 10px 10px 40px 0px #000000;}- adds a "shadow" around top/right/bottom/left sides of the box to the specified extent of pixels; last value in the attribute is a color that should be added through the numeric code.

Background

  • Adding multiple images to a background
    • background-image: url('name_of_image'.png), url('xxx.png'), etc.; - extra images are added by dividing the image urls with a commna (,).
    • background-position: 50px 50px, 400px 50px, etc.; - the position is altered in this format: (distance_from_left)px (distance_from_top)px, etc.
    • background-repeat: no-repeat;
  • Background origin
    • background-origin: padding-box/border-box/content-box; - choice of either include or exclude the border
  • Background sizing
    • background-size: 50% 50%; - increase the size of the background image's top/bottom by 50% and left/right by 50%

Fonts

CSS3 allows you to import specific fonts which may not be available by default. Via a special method "@font-face", that font is added to the web page and then can be normally used as:

p {font-family: 'Name of the imported font';}

The full method is shown and explained step by step can be viewed if you follow the link below:

A rather interesting than useful feature:

Opacity

  • Whole Elements
    • #id {opacity: 0.3} - makes a fading away effect on the element.
  • Color
    • #id {border-color: rgba(xxx.x, xxx.x, xxx.x, xxx.x);} - makes a fading away effect only on the specified part of the element.
