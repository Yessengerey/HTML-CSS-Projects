Links

Link or rather HyperLink is a very useful tool. It does not only allow you to organize the created web pages in a convenient enough way but also allows you to provide roots for more information on the web. HyperLinks are also called Anchors or just Links.

As defined in the W3C specification - "A link is a connection from one web resource to another"

Links are created using an inline tag <a>. The href attribute specifies the location to link to. An example of a link you can see below:

<a href="destination">a link</a> »»» a link

Common Link Types

There are several common link types that are widely used, list of which is shown below:

  • Link to another website
    • In this case the href attribute has to be the full absolute URL of the website you intend this link to take you to.
    • <a href="http://www.w3.org/">W3C main page</a>
  • Link to the other pages on your site (through local file system)
    • The href should have the name and the type of the file it is intended to take you to. That web document should be stored in the same folder as the page from where the link is originated.
    • <a href="0TutorialMainPage.html">QLT - main page</a>
  • Link to the parts of a page
    • The href should be pointed at a specific fragment of the destination document that is identified with the id
    • <a href="0TutorialMainPage.html#navigation">QLT - main page (navigation menu)<a>
  • Links within the page
    • The href will have to be referring to the tag that will contain the specified id
    • <a href="0TutorialMainPage.html#an_id">QLT - main page</a>
  • Link to a web page which is located in a different folder relative to the folder where the source page is
    • The href will have to have sections, names of the folder, divided by a backslash.
    • <a href="folder_name/0TutorialMainPage.html">QLT - main page</a>
